# ML Algorithms from Scratch (NumPy)

Реализация ключевых алгоритмов машинного обучения на чистом NumPy в рамках курса Машинного обучения и анализа данных ФПМИ МФТИ

## Реализованные алгоритмы:
### 1. Линейные модели и Оптимизация:

 — [Линейной](linear_model.py) и логистической регрессии

 — Различные стратегии [градиентного спуска](descents.py) (GD, Heavy ball, NAG, Newton)

 — Математическая реализация функций потерь (MSE, Log-Loss, Hinge Loss)

### 2. SVM:

 — Линейный [SVM](linear_svm.py)

 — [SVM с RBF ядром](kernel_svm.py)

### 3. Ансамблирование и Сэмплирование:

 — Реализация [бэггинга](bagger.py)

 — Реализация [бустинга](booster.py)

 — [Сэмплер](sampler.py) бутстреп-выборок

### 4. Обучение без учителя:

 — Реализация [k-means](k_means.py)

 — Реализация [k-means++](k_means++.py)

 — Реализация [DBSCAN](DBSCAN.py)

 — Реализация [классического и ядерного PCA](pca.py)

Вычисления оптимизированы через матричные операции, используется только NumPy
